HTML

Parts of an HTML element

Element: elements can be nested.

	<someElement>

Tag: tags are not case sensitive.

	<openingTag> </closingTag>

Attribute: if it has no value it's called boolean attribute.

	<someElement someAttribute=someValue>

Special characters: (< = &+lt+;) (> = &+gt+;) (" = &+quot+;) (' = &+apos+;) (& = &+amp+;)

Text visual

Bold: b or strong

Italic: i or em

Span and i: span, inline-level element, used mostly for style grouping

Input: has a boolean attribute that has no value (<input disabled>, same as <input disabled="true">)

Image: is a void element because it consists only of a single opening tag. 

	<img src="someLink" alt="alternative text">

Anchor: <a href="someLink" title="Hover text">Some text</a>, turns text into links.

This is an anchor text that is a link. If you wrap a heading or an image with an anchor, you can create headings and images that are links.
